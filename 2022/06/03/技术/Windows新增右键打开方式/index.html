Windows新增右键打开方式

本文最后更新于：3 个月前

一、添加右键菜单

1.注册表打开

2.修改项

参考：https://zhuanlan.zhihu.com/p/428704111
计算机\HKEY_CLASSES_ROOT\*\shell # 右击文件时弹出的菜单
计算机\HKEY_CLASSES_ROOT\Directory\shell # 右击文件夹时弹出的菜单
计算机\HKEY_CLASSES_ROOT\Directory\Background\shell # 右击文件夹空白处时弹出的菜单

3.示例

1) 右击文件时弹出的菜单

  1. Win+R 打开运行，输入regedit，打开注册表，找到”计算机\HKEY_CLASSES_ROOT\*\shell”分支，如果没有shell分支，则在*下点击右键，选择“新建－项”，建立shell分支。

  2. 在shell下新建”VSCode”项，在右侧窗口的“默认”双击，在数据里输入“Edit with Visual Studio Code”。这是右键上显示的文字，可以随便写，只是为了方便查看。

  3. 在“VSCode”下再新建Command项，在右侧窗口的“默认”键值栏内输入程序所在的安装路径：

    D:\Program Files\Microsoft VS Code\Code.exe “%1”

    或者 “D:\Program Files\Microsoft VS Code\Code.exe” “%1”

    其中的%1表示要打开的文件参数。

  4. 配置缩略图。在VSCode项上新建可扩充字符串值，命名为Icon，双击，把”D:\Program Files\Microsoft VS Code\Code.exe”放进数据就可以了。

  5. 关闭注册表，即可生效。

2) 右击文件夹时弹出的菜单

  1. Win+R 打开运行，输入regedit，打开注册表，找到”计算机\HKEY_CLASSES_ROOT\Directory\shell”分支

  2. 同上面的2一样，数据内的值为“Open with Visual Studio Code”。

  3. 在“VSCode”下再新建Command项，在右侧窗口的“默认”键值栏内输入程序所在的安装路径：

    D:\Program Files\Microsoft VS Code\Code.exe “%V”

  4. 配置缩略图。在VSCode项上新建可扩充字符串值，命名为Icon，双击，把”D:\Program Files\Microsoft VS Code\Code.exe”放进数据就可以了。

  5. 关闭注册表，即可生效。

3) 右击文件夹空白处时弹出的菜单

  1. Win+R 打开运行，输入regedit，打开注册表，找到”计算机\HKEY_CLASSES_ROOT\Directory\Background\shell”分支

  2. 同上面的2一样，数据内的值为“Open with Visual Studio Code”。

  3. 同上，把 %1 改为%V，例如：

    D:\Program Files\Microsoft VS Code\Code.exe “%V”

4、5同上，完全一样
